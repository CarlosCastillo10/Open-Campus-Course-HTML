Un sistema binario se representa con:

    1 y 0
    Cantidades numéricas decimales
    Códigos alfa-numéricos.

El sistema de numeración octal contiene los siguientes símbolos:

    {0,1,2,3,4,5,6}
    {1,2,3,4,5,6,7,8}
    {0,1,2,3,4,5,6,7}

¿Qué hago cuando tengo una suma con una parte entera y otra decimal, por ejemplo 101.1+10?

    Realizo la suma normalmente y ubico la coma en donde corresponde.
    Realizo la suma, y corro la coma sumando el número de decimales de cada cifra.
    Los decimales desaparecen al momento de hacer la suma.

¿Un sistema de numeración es?

    Formulación de un mensaje a través de las reglas o normas de un código o lenguaje predeterminado
    Conjunto de símbolos para representar cantidades numéricas
    Conjunto alfanumérico para representar números
    Conjunto ordenado de normas y procedimientos que sirven para representar números

¿La notación expandida de un número es?

    Escribir un número de tal manera que se muestra el valor de cada dígito
    Escribir un número de tal manera que se puedan distinguir la posición de cada dígito
    Escribir un número de tal manera que se conozca su equivalente en número decimal.
