¿QUÉS ES UNA BASE DE DATOS?

Existen diversas definiciones de base de datos, una de las más comunes es:

Una base de datos es una colección de información un “almacén” que nos permite guardar grandes cantidades de información de forma organizada para que luego podamos encontrar y utilizar fácilmente. 

Otra definición bastante aceptada nos dice:

Una base de datos es una serie de datos organizados y relacionados entre sí, los cuales son recolectados y explotados por los sistemas de información de una empresa o negocio en particular.
