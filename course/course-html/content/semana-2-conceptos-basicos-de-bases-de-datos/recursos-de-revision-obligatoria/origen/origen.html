ORIGEN DE LAS BASES DE DATOS

  • El ser humano siempre ha tenido la necesidad de guardar información. 
  • Los orígenes de las bases de datos se remontan a la Antigüedad donde ya existían bibliotecas y toda clase de registros (se utilizaban para recoger información sobre las cosechas y censos).
  • El uso de las bases de datos se desarrolló a partir de las necesidades de almacenar grandes cantidades de información o datos.

Se recomienda observar el video que se muestra a continuación para comprender el origen de las base de datos y su evolución a través de los años.
