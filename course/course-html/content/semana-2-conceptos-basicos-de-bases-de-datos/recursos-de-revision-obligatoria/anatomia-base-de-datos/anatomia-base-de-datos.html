¿anatomía de una UNA BASE DE DATOS?

Una base de datos, esta compuesta por una o más tablas:

  • Tablas:  son una colección de información relacionada; mantiene junta  esta información del mismo modo que hace la carpeta de cualquier archivador. Una tabla es una colección de registros.
  • Registro: es la información relacionada con una persona, producto o evento.
  • Campo: es cada parte concreta de la información contenida en un registro. El tipo de información de un campo viene determinado por su tipo de campo o tipo de dato.

Puede ser:

  • Campo numérico
  • Campo de texto
  • Campo fecha
