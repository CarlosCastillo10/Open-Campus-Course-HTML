Software: definición y clasificación

Se define como software la conjunto de instrucciones  que permiten controlar los dispositivos físicos que conforman el computador para que realicen las operaciones que tienen destinadas. Constituye una parte lógica o intangible.

El software se puede clasificar según varios criterios, pero en este curso se indicará la clasificación más general, que distingue dos tipos:

  • Software del Sistema
  • Software de Aplicación

El siguiente recurso, le permitirá distinguir el software del sistema, obsérvelo las veces que considere necesario, tome nota de las ideas más importantes. Estamos más familiarizados con el hardware que con el software por tanto para este tema le recomiendo poner mucha atención en la subclasificación que se indica del software del sistema, su función y ejemplos. El recurso es tomado de la Universidad Politécnica de Valencia.
